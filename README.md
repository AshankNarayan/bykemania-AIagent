# BykeMania AI Operations Agent

A production-ready FastAPI backend for BykeMania operations.

This backend provides:

- Natural-language AI chat for fleet and operations queries
- Live fleet availability responses
- Department-wise alert generation
- Alert history storage
- Dashboard APIs
- Scheduler support
- PostgreSQL logging
- API key security
- Production hardening with health checks, request IDs, timeouts, and rate limiting
